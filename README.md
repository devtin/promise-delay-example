## Usage

Just run it using npx:
```sh
$ npx https://github.com/devtin/promise-delay-example
```

Or clone the repo.
```sh
$ git clone https://github.com/devtin/promise-delay-example
$ cd promise-delay-example
$ yarn && yarn start
# or using npm
$ npm install && npm start
```
